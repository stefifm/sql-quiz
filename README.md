# SQL Quiz

Bienvenido a SQL Quiz, un pequeño juego de preguntas y respuestas que te permitirá poner a prueba tus conocimientos en SQL. Inspirado en el juego [Javascript Quiz](https://javascriptquiz.com/) y en la clase de React de [@midudev](https://github.com/midudev/aprendiendo-react/tree/master/projects/13-javascript-quiz-con-zustand)

## ¿Cómo jugar?
Se trata de 10 preguntas y cuatro opciones. Tendrá que seleccionar la respuesta correcta y en el momento sabrá si acertó o no. Al final del juego, se mostrará su puntuación.

## ¿Cómo se hizo?
- Se utilizó React + Vite para la creación de la aplicación.
- JSON Bin para almacenar las preguntas y respuestas.
- Se utilizó Zustand para el manejo del estado global de la aplicación.
- Material UI para los estilos.
- React Syntax Highlighter para mostrar los códigos SQL.
- TypeScript para la tipificación de los datos y escritura de código.
